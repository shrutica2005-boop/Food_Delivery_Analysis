Food Delivery Data Analysis
This repository contains a Jupyter Notebook developed using Google Colab for analyzing a food delivery dataset using Python and Pandas.

Overview
The notebook performs:
Loading and inspection of the dataset
Merging multiple datasets (orders, users, and restaurant details)
Data filtering and aggregation
Analysis of revenue, user activity, membership-based trends, and restaurant ratings

Tools & Technologies
Python
Pandas
Google Colab
Jupyter Notebook

Files
Food_Delivery_Analysis_shruti_rajopadhye.ipynb – Main analysis notebook
final_food_delivery_dataset.csv – Dataset used for analysis
